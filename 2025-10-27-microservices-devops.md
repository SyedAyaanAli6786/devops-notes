# Class Notes – Microservices & DevOps (Today’s Session)
## Key Takeaways

Microservices are small, independent services that can be developed and deployed separately

REST APIs enable communication between services

Stateless services do not store data internally

External logging helps in efficient monitoring and debugging

Load balancers distribute traffic across services

DNS translates service names into IP addresses

DevOps focuses on automation and collaboration between teams

CI/CD pipelines ensure faster and more reliable deployments

Hands-on practice is essential to fully understand these concepts
